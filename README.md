# Auto Llamarist Luani — Atelier i Llamarinës 🏆🚗

Faqe interneti premium (versioni "atelier") për **Auto Llamarist Luani** — dizajn luksoz me
ngjyra ari/qymyri, tipografi serif, kursor i personalizuar dhe animacione scroll.

Ndërtuar me **Next.js 15**, **Tailwind CSS** dhe **Framer Motion**.

## Zhvillimi
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # build statik për GitHub Pages
```

## Përshtatja
- Numri, email, adresa, orari, markat, fotot → `lib/site.ts`
- Shërbimet → `components/Services.tsx`
- Logot dhe fotot reale → `public/logos/` dhe `public/work/`

> Imazhet e veturave janë vendmbajtëse nga një CDN (me fallback SVG). Zëvendësohen me foto reale.
